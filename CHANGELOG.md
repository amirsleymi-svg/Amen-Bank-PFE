# Changelog

All notable changes to Amen Bank are documented here.
Format: [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)

## [1.0.0] — 2024-01-01

### Added
- Full Angular 17 SPA with lazy-loaded routes
- Spring Boot 3.2 REST API with JWT + TOTP authentication
- FastAPI chatbot with Redis session memory
- Google Authenticator 2FA on all sensitive operations
- Transfer simple with TOTP modal confirmation
- Batch CSV transfer processing
- Credit simulation with full amortization table
- KYC onboarding with admin approval workflow
- Admin panel: KYC, users, credits, audit logs
- Profile page: TOTP setup/disable, password change
- 9 Thymeleaf email templates
- Docker Compose full stack
- GitHub Actions CI/CD (6-job pipeline)
- Prometheus + Grafana monitoring
- Cypress E2E tests + JUnit5 + pytest
